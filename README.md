# GitHub Issue Dashboard

Automated dashboard showing open issues across all my GitHub repositories.

## 📊 [View Dashboard](DASHBOARD.md)

## Features

- 🔄 Auto-updates daily via GitHub Actions
- 📈 Shows all repos with open issues
- 🏷️ Displays issue labels and metadata
- ⏰ Shows issue age and last update time
- 🔗 Direct links to each issue

## How It Works

1. GitHub Actions runs daily at 6 AM UTC
2. Script fetches all repositories and their open issues
3. Generates `DASHBOARD.md` with formatted data
4. Commits and pushes changes automatically

## Manual Update

To manually update the dashboard:

```bash
./scripts/generate-dashboard.sh
```

## Setup

This dashboard uses GitHub CLI (`gh`) to fetch data. The GitHub Actions workflow uses `GITHUB_TOKEN` automatically provided by GitHub Actions.
